# 🎨 DaisyUI AI Chat Bot Starter Design System

Welcome to the **DaisyUI AI Chat Bot Starter** design system! This comprehensive component library is built with modern web technologies and follows atomic design principles to create beautiful, accessible, and performant user interfaces.

## 🚀 Features

- **🤖 AI-Powered**: Built for AI chat applications with streaming support
- **🎨 Modern UI**: Beautiful components with DaisyUI styling
- **📱 Responsive**: Mobile-first design with adaptive layouts
- **🎭 Smooth Animations**: Motion.create components throughout
- **🧩 Atomic Design**: Scalable component architecture
- **⚡ Type-Safe**: CVA-powered component variants
- **♿ Accessibility**: WCAG compliant with semantic HTML
- **🌙 Theme Support**: 32 built-in DaisyUI themes

## 🏗️ Architecture

Our design system follows the **Atomic Design** methodology:

### 🔬 Atoms

Basic building blocks that can't be broken down further:

- **Button** - Interactive elements with multiple variants
- **Input** - Form inputs with validation states
- **Link** - Navigation elements with Next.js integration

### 🧬 Molecules

Simple combinations of atoms:

- **Footer** - Copyright and author information
- **SearchBox** - Input with search functionality
- **FormField** - Input with label and validation

### 🦠 Organisms

Complex components made of molecules and atoms:

- **Navbar** - Navigation bar with mobile drawer
- **ChatArea** - AI chat interface with streaming
- **Sidebar** - Navigation sidebar with collapsible sections

### 📄 Templates

Page-level layouts that define structure:

- **MainLayout** - Primary application layout
- **AuthLayout** - Authentication pages layout
- **ChatLayout** - Chat-specific layout

## 🎯 Component Patterns

### CVA Variants

All components use Class Variance Authority for type-safe variants:

```typescript
const buttonVariants = cva("btn", {
  variants: {
    variant: {
      primary: "btn-primary",
      secondary: "btn-secondary",
    },
    size: {
      sm: "btn-sm",
      md: "btn-md",
      lg: "btn-lg",
    },
  },
  defaultVariants: {
    variant: "primary",
    size: "md",
  },
});
```

### Motion.create Pattern

All atoms use `motion.create()` for consistent animations:

```typescript
const MotionButton = motion.create("button");

// Usage with motion props
<Button whileHover={{ scale: 1.02 }} whileTap={{ scale: 0.98 }}>
  Animated Button
</Button>
```

### File Naming Convention

- **Client Components**: `ComponentName.client.tsx`
- **Server Components**: `ComponentName.tsx`
- **Types**: `ComponentName.types.ts`
- **Stories**: `ComponentName.stories.tsx`
- **Barrel Exports**: `index.ts`
